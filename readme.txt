--Hanged--

Descripción del programa:
=> Simulación del clásico juego "Ahorcado" para la consola del sistema operativo.

=> Adivina la palabra oculta con un número de intentos límitados.

Instrucciones de ejecución:
=> Desde la consola del sistema sitúate en el directorio "dist" con el comando cd + "ruta del directorio"
 y ejecuta el comando "java -jar Hanged.jar".

Tecnologías utilizadas:
=> Java

Versión 1.0.
